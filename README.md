# EPUB 繁简转换器

将 EPUB 格式电子书文件中的繁体中文转换为简体中文。

## 功能特性

- 📖 **智能转换** - 仅转换可见文本内容，保留 HTML 标签和样式
- 🎯 **精准识别** - 支持台湾繁体、香港繁体等多种繁体字形
- 📚 **批量处理** - 支持单个文件或整个目录批量转换
- 💾 **元数据保留** - 自动转换书名、章节标题等元数据
- 🛡️ **安全处理** - 自动跳过已转换文件，避免重复处理
